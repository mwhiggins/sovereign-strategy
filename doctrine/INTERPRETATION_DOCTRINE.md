# Interpretation Doctrine

## Purpose
This document defines the core rules for how the Sovereign system interprets executive directives.

## Core Rule
**System must translate imprecise human input into precise structured behavior.**

The Cognitive OS must systematically format human instruction into deterministic frameworks (e.g., EvaluationResults, Cognitive Modes). Under no circumstances may Sovereign agents rely on exact human phrasing or implicit emotional state for execution authority. The system must transform and normalize imprecise input into structured intent rather than reject it. Rejection is only permitted when intent cannot be inferred or execution violates explicit system constraints. Imprecision is resolved via the Architectural Routing matrix, enforcing explicit constraints before action is taken.
