# Interpretation Doctrine

## Purpose
This document defines the core rules for how the Sovereign system interprets executive directives.

## Core Rule
**System must translate imprecise human input into precise structured behavior.**

The Cognitive OS must systematically format human instruction into deterministic frameworks (e.g., EvaluationResults, Cognitive Modes). Under no circumstances may Sovereign agents rely on exact human phrasing or implicit emotional state for execution authority. Imprecision is resolved via the Architectural Routing matrix, enforcing explicit constraints before action is taken.
